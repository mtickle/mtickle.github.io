

## Current Projects

### [UnscrambleJS](https://mtickle.github.io/UnscrambleJS/)
An unscramble word game written in jQuery with various other plugins and Bootstrap. The game features the ability to lock a letter in place, shuffle and time addition. Eventually, the enhancements will be in a marketplace where the player will have to buy them with coins earned while solving puzzles. I'm currently working through some issues with the letter lock and shuffle.

### [PoliceReports](https://mtickle.github.io/PoliceReports/)
This technology demonstration pulls a list of recent Raleigh Police Department events (as JSON) from Open Data Raleigh and displays them on a webpage. I'm using jQuery to pull and clean up the JSON before processing, as well as the Google Static Maps API to show the location of the event. Bootstrap wraps it all up and makes it look pretty. 

### [Proverbs Verse of the Day](https://mtickle.github.io/VerseOfTheDay/)
Demonstration, using jQuery, of displaying a random verse from Proverbs. While I had initially decided to use a Bible API to get the verses, I realized that the Bible really doesn't change that often, and a flat file would do.
